    install and use dropbox on archlinux

    Mario Long

    2018-09-07 12:18

    Comments

    1 install

    因 archlinuxfr 中也有 dropbox，為了不和它衝突，特別加上 aur/dropbox 

    yaourt -S aur/dropbox

    2 config dropbox.service

    2.1 setup environment variables

    為了正確顯示，要加上以下命令： 

    Environment=DISPLAY=:0
Environment=LC_ALL=zh_TW.UTF-8 LANG=zh_TW.UTF-8 QT_STYLE_OVERRIDE=GTK+

    2.2 set Restart=no in dropbox.service

    根據 Dropbox kill/restart loop when started at boot via systemd，在 dropbox.service 中，特別加上 

    Restart=no

    2.3 final dropbox.service

    [Unit]
Description=Dropbox for systemd/user

[Service]
Environment=DISPLAY=:0
Environment=LC_ALL=zh_TW.UTF-8 LANG=zh_TW.UTF-8 QT_STYLE_OVERRIDE=GTK+
Type=simple
ExecStart=/usr/bin/dropbox
ExecReload=/bin/kill -HUP $MAINPID
KillMode=process
Restart=no

[Install]
WantedBy=default.target

    3 start dropbox

    最後執行 

    systemctl start dropbox --user

    or/and 

    systemctl enable dropbox --user

    即可。
