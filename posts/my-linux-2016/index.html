    這一年，幾乎沒有大的改變。

    我的 linux 平台

    整理一下目前在 linux 下，目前還在用的工具。試過後，不用的就不必放進來。

    目的是，如果有一天要重新安裝系統時，能快速完成，回到可工作狀態。

    作業系統

    Install Arch Linux : 接觸 linux 後，就只用過這個發行版。喜歡它的 rolling update，一直是用最新版。

    xterm

    fish: current shell

    桌面

    lxqt : 簡單好用的「台灣之光」

    我的桌面組合 lxqt+xmonad+taffybar ，著重於工作效率的平舖式視窗管理

    Show volume icon on systray

    字型相關

    Rime 中州韻輸入法引擎

    網路工具

    google-chrome

    youtube-dl: 超強的 youtube downloader

    qbittorrent: BT client

    wifi-ap: wireless AP 設定 （hostapd+netcfg/netctl）

    影音平台

    mm=mpv+feh 好用的影音播放器

    kodi: 家庭劇院

    ffmpeg 使用心得與整理 ，有了這個，可以應付影音轉檔的需求。

    DVD backup ，備份 DVD 吧！現在都是看 BD，實用性已經不高。

    aegisub: 字幕處理。

    mpd+mpc+ncmpcpp: 聽音樂的好朋友。 enable mpd with systemd

    kid3: 修改音樂檔的 tags

    lightworks: 非線性影音剪輯系統

    生活回憶

    feh 小巧好用的圖片閱覽器

    mm=mpv+feh: 自己寫吧，找不到真正喜歡的。比較接近需求的是 gwenview，又有無法解決的問題，且太過笨重。
