    Mario Long

    2014-02-02 18:51

    Comments

    lightworks連結

    Description This is a Lightworks PKGBUILD for Arch Linux.

    Howto:

    First Download lightworks-11.5-1-x86_64.pkg.tar.xz

    Then open your Terminal and install it: $ sudo pacman -U lightworks-11.5-1-x86_64.pkg.tar.xz

    Register your User on http://www.lwks.com/ and launch Lightworks.

    Tested on Antergos, Enjoy.
