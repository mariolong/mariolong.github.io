    Install Arch linux (2017)

    Mario Long

    2017-11-20 17:00

    Comments

    • linux

    原本的 Beginners' Guide 改成了 Installation guide ，文件結構比較清楚，但是保證不適合初次安裝 archlinux 的人閱讀，也不適合一步步之執行。

    這次安裝，把目前系統應有的指令再記録下來，以備參閱。與上次安裝時所用的指令，差異不大。

    1 製作開機片（CD、USB、硬碟）啓動

    從 Arch linux Downloads 下載 archlinux.iso，建議用 bt 方式下載比較快。

    參考 USB flash installation media 製作開機片 : 

    dd bs=4M if=/path/to/archlinux.iso of=/dev/sdx status=progress && sync

    2 啟動 ArchLinux 安裝程序

    重新以 USB 開機，選擇從主選單中選擇 "Boot Arch linux" 並按 <enter>，系統將加載並給出登錄提示，自動以 'root' 登錄。

    3 建立網絡連接

    ping 8.8.8.8 看看有没有網路，没有網路就啓動 dhcpcd: 

    systemctl start dhcpcd

    若要用 wifi 則用: 

    wifi-menu -o profile-name

    ping 一下看看網路連上了嗎？

    4 準備磁碟機

    因為先安裝 windows 10，已經將磁碟分割好了。

    4.1 檢查分區狀況

    lsblk -f

    4.2 格式化

    格式化目標 partition，免得有不可預期的狀況 

    mkfs.ext4 -L LABLE /dev/sdax

    4.3 掛載分區

    先掛載 root 區 : 

    mount /dev/sdaX /mnt

    再掛載其他分區，/home 最好獨立分區。 : 

    mkdir /mnt/home
mount /dev/sda1 /mnt/home

    4.4 建立 swap

    現在的 RAM 都很大，這個可以不用裝了。目前安裝了 16G 的 RAM。

    4.4.1 建立 swap file

    如果真的要裝 swap，可用檔案取代，方便好管理。 : 

    fallocate -l 16G /swapfile
sudo chmod 600 /swapfile
mkswap /swapfile
sudo mkswap /swapfile
swapon /swpfile
sudo swapon /swapfile
sudo nano /etc/fstab

    在 fstab 中加入 : 

    /swapfile none swap defaults 0 0

    4.4.2 Remove swap file

    如果不用 swap，刪了就是。

    Warning: swap is managed by systemd and will be reenabled by it after some time. To remove a swap file, the current swap file must be turned off. : 

    swapoff -a
rm -f /swapfile

    5 配置安裝的環境

    5.1 DNS setup (可不做)

    nano /etc/resolv.conf

    nameserver 168.95.1.1
nameserver 8.8.8.8

    5.2 選擇安裝鏡像

    安裝前需要編輯 /etc/pacman.d/mirrorlist, 將最想使用的鏡像放到前面： 

    nano /etc/pacman.d/mirrorlist

    nano 簡單指令 : 

    Alt-6: copy line
C-u:   paste
C-x:   exit

    原則上，改過 mirrorlist 就執行以下指令(可不做)： : 

    pacman -Syy

    6 安裝基本系統

    pacstrap -i /mnt base base-devel

    if pacman complains that error: 

    failed to commit transaction (invalid or corrupted package)

    run the following command: 

    pacman-key --init && pacman-key --populate archlinux

    6.1 生成 fstab

    這個一定要做，不然後頭會很麻煩。 : 

    genfstab -U -p /mnt >> /mnt/etc/fstab
nano /mnt/etc/fstab   # 檢查一下 fstab

    6.2 Chroot 到新系統，並作基本配置

    arch-chroot /mnt /bin/bash

    6.3 配置 pacman

    nano /etc/pacman.conf

    爲了安裝 yaourt，到最後空白處加上： : 

    [archlinuxfr]
SigLevel = Never    # add: 2013/04/28
Server = http://repo.archlinux.fr/$arch

    一般使用者用 [core], [extra] 與 [community]。

    6.4 更新系統，並安裝 yaourt

    pacman -Syu
pacman -S yaourt

    6.5 配置系統 Locale

    nano /etc/locale.gen

    6.5.1 找到

    en_US.UTF-8 UTF-8
zh_TW.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8

    把以上全部打開。

    6.5.2 使 locale 生效

    locale-gen    #使更改生效運行

    6.5.3 確認一下 locale

    locale -a     #確認一下開啟的 locale
locale        #目前的 locale

    6.5.4 全域 locale 設定：對整個系統有效

    nano /etc/locale.conf

    修改 LANG, 主控台用英文，否則出現方塊字，是看不懂它的意思。等進 X 後再改成中文。 

    export LANG=en_US.UTF-8

    6.6 更改 console 字型

    把字改大一點，眼睛比較不吃力。先安裝字型，再設定字型。 

    pacman -S terminus-font
setconsolefont ter-128n

    The FONT variable in /etc/vconsole.conf is used to set the font at boot, persistently for all consoles.1

    nano /etc/vconsole.conf

    加上 

    FONT=ter-128n
FONT_MAP=8859-1

    ISO 8859-1，正式編號為 ISO/IEC 8859-1:1998，又稱 Latin-1 或「西歐語言」2

    6.7 配置系統時區

    可用的時區位於目錄 /usr/share/zoneinfo/ 下，可以 ls 一下。用以下指令配置系統時區： 

    ln -s /usr/share/zoneinfo/Asia/Taipei /etc/localtime

    6.8 時間調整

    為了與 windows 雙重開機 (dual boot)，最好做以下動作：

    1. 修改 windows 的登錄檔，用 regedit 加上一個 DWORD(32 bits) 的機碼 RealTimeIsUniversal，將其值設為 1，位置如下: 

      HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\TimeZoneInformation\RealTimeIsUniversal
    2. 在 windows 中關掉「網路自動對時」的功能。

    當然，以上的動作要回到 windows 才能執行。

    在 linux 中，只需執行： 

    timedatectl set-local-rtc false
timedatectl set-ntp true

    並將 utc 時間寫入 BIOS 中即可。 

    hwclock --systohc --utc

    6.9 設置主機名稱

    nano /etc/hostname

    加入 

    hostname

    7 再一次設定網路

    簡單啓動 dhcpcd : 

    systemctl enable dhcpcd

    7.1 WIFI

    用 netctl 啓動 wifi

    7.2 ADSL

    如果要用 ADSL，則這個很重要，不然重開機時，就會沒網路可用。

    install the network card that is supposed to be connected to the DSL-Modem into your computer. After adding your newly installed network card to the modules.conf/modprobe.conf install therp-pppoe package : 

    pacman -S rp-pppoe
pppoe-setup

    run the pppoe-setup script to configure your connection After you have entered all the data, you can connect and disconnect your line with : 

    systemctl start adsl

    and: 

    systemctl stop adsl

    respectively. The setup is usually easy and straightforward, but feel free to read the manpages for hints. If you want to automatically 'dial in' at boot, issue command : 

    systemctl enable adsl

    or 

    systemctl disable adsl

    to remove auto 'dial in' at boot.

    8 mkinitcpio

    增加 lvm2 到 hooks 後，執行 mkinitcpio : 

    nano /etc/mkinitcpio.conf

    修改 

    HOOKS=(... udev ... block lvm2 filesystems ...)

    再執行 

    mkinitcpio -p linux

    9 Install and configure a bootloader

    Install the grub-bios package and then run grub-install: 

    pacman -S grub-bios  ## 安裝 grub-bios
grub-install --target=i386-pc --recheck /dev/sda  ## 安裝到 boot 區

    9.1 自動尋找其它的 OS

    如果有其他的 OS，這個就很重要。 

    pacman -S os-prober

    或是直接在 /etc/grub.d/40\custom 加上以下指令： 

    menuentry "Windows 10" --class windows --class os {
  insmod ntfs
  search --no-floppy --set=root --fs-uuid $your_uuid_here$
  ntldr /bootmgr
}

    9.2 intel CPU 要安裝 microcode

    這步要在執行 grub-mkconfig 自動產生 grub.cfg 之前做： 

    pacman -S intel-ucode

    9.3 自動產生 grub.cfg

    這個一定要做，不然重開機時，就等著進 grub shell，很麻煩。 

    grub-mkconfig -o /boot/grub/grub.cfg

    10 重啓

    exit
umount /mnt
reboot

    10.1 確認網路連線

    root 重啓後，ping 一下看看網路連上了嗎？如果沒有，手動連線。 

    ping 8.8.8.8
systemctl start dhcpcd

    或 

    netctl start <wifi-profile-name>

    10.2 update system（可不做）

    pacman -Syu

    11 post-installation

    11.1 設置 Root 密碼並創建一般用戶

    passwd
useradd -m -s /bin/bash  -g users -G video,storage,optical,lp,scanner,games,wheel username
passwd username

    11.2 安裝 Sudo

    pacman -S sudo
EDITOR=nano visudo

    允許 wheel 用戶組成員無密碼使用 sudo： 

    visudo

    輸入 

    %wheel ALL=(ALL) NOPASSWD:ALL

    再來是設定 sudo，隨便找一個空白行加上： 

    username ALL=(ALL) ALL

    11.3 登出，並以新的用戶名稱登入

    11.4 設定聲音 alas

    安裝音效驅動，直接用 alas，不要考慮太多。 

    yaourt -S alsa-utils

    將帳號加入音效使用權： 

    sudo gpasswd -a username audio

    alsa 預設是靜音，以 alsamixer 手動打開聲音： 

    alsamixer

    或者命令列直接開啓： 

    amixer sset Master unmute

    測試一下有沒有聲音： 

    speaker-test -c 2

    11.5 安裝 x11

    pacman -S xorg-{server,apps} mesa

    因為要用 xlogin，所以 xorg-xinit 可以不用裝了。

    11.6 裝顯卡驅動

    11.6.1 ATi 用戶：

    直接裝開源驅動 

    pacman -S xf86-video-amdgpu

    或是用舊的 

    pacman -S xf86-video-ati

    11.6.2 intel 用戶：

    pacman -S xf86-video-intel libav-intel-driver

    11.7 安裝字體

    • ttf-liberation: 修正部份 PDF 字型，此為 google-chrome 新的依賴。
    • ttf-hanazono: 花園明朝體，罕見字
    • ttf-tw: 教育部標準楷宋體
    • ttf-symbola: Font for unicode symbols.
    yaourt -S ttf-tw ttf-dejavu ttf-liberation ttf-hanazono ttf-symbola
yaourt -S adobe-source-{code,sans,serif}-pro-fonts adobe-source-han-{sans,serif}-{cn,jp,kr,otc,tw}-fonts

    11.7.1 ttf-twcns-fonts: 中文全字庫

    參考 http://samwhelp.github.io/book-ubuntu-qna/read/howto/configure-font/font/cns11643/install 

    wget -c http://www.cns11643.gov.tw/AIDB/Open_Data.zip
unzip Open_Data.zip

    Copy all *.ttf files to ~/.local/share/fonts, and execute fc-cache. Run fc-list to check.

    11.8 讓 X 可以顯示中文

    nano ~/.xinitrc

    export LANG=zh_TW.UTF-8
export LC_ALL="zh_TW.UTF-8"

    11.9 將右邊的數字鍵盤預設爲方向鍵

    yaourt -S numlockx
nano ~/.xinitrc

    增加 

    numlockx &

    11.10 安裝桌面

    lxqt 不全裝，目前只用 lxqt-policykit-agent 和 lxappearance.

    用 install haskell application 這篇安裝 xmonad 和 taffybar.

    11.11 theme

    Using Breeze theme and Flat Remix icon theme: 

    yaourt -S breeze{,-gtk} flat-remix-git lxappearance adwaita-icon-theme

    進 lxappearance 設定 theme and icon theme

    11.12 安裝中文輸入法

    沒有其它選擇，就用 fcitx。 

    yaourt -S fcitx-im fcitx-configtool fcitx-chewing

    11.13 install google chrome

    yaourt -S google-chrome

    11.14 安裝文字編輯器

    現在已經沒有別的選擇了，只能用 emacs 這個神的編輯器，因為她實在是太強大了，讓我不忍放棄她。 

    yaourt -S emacs

    可用的系統成型了，但是還是要再參考 Post-installation of Archlinux 有許多需要再調整的。
