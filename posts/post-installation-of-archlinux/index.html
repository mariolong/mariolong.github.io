    首先依 Install Arch linux (2017) 將基本系統裝好，接著參考 General recommendations - ArchWiki 記錄下對我而言必要的軟體及簡單設定，讓日常進入可工作狀態。

    1 haskell

    安裝的方法參看 Install Haskell Applications

    2 桌面及外觀

    2.1 theme

    yaourt -S breeze{,-kde4,-gtk} flat-remix-git lxappearance

    Using Breeze theme and Flat Remix icon theme

    2.2 lxappearance

    用 lxappearance 修改 gtk 程式的外觀。

    尤其是 google chrome 書籤列的字體，可能太小。這時即可用 lxappearance 調整字型和大小。

    2.3 oblogout and screen lock

    https://wiki.archlinux.org/index.php/Oblogout

    oblogout is an optional, configurable logout script that presents a graphical interface (i.e. buttons) to cancel, logout, restart, shutdown, suspend, hibernate, and lock the screen. 

    yaourt -S oblogout

sudo nano /etc/oblogout.conf

    修改 [looks] 中的 buttons，以顯示按鈕。 

    [looks]
buttons = cancel, restart, shutdown, lock

    修改 [shortcuts] 中不用的命令刪掉。 

    [shortcuts]
cancel = Escape
shutdown = S
restart = R
#suspend = U
#logout = L
lock = K
#hibernate = H

    修改 [commands] 中的命令。 

    [commands]
lock = slock

    5.1.1 issues

    error message: 

    /usr/bin/xdg-open: line 922: pcmanfm: command not found

    fix this problem: 

    sudo ln -s /usr/bin/pcmanfm-qt /usr/bin/pcmanfm
