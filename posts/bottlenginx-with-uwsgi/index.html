    bottle+nginx with uwsgi

    Mario Long

    2014-02-08 13:35

    Comments

    安裝 uwsgi

    sudo pip uwsgi

    先寫一個小小的程式，測試一下。

    #!/usr/bin/env python
# -*- coding: utf-8 -*-

from bottle import route, app

@route('/')
def a():
    return( "Hello world" )

application = app()

    啟動 uwsgi

    uwsgi --http :8001 --wsgi-file wsgi-test.py --chdir /path/to/wsgi-test.py

    進 browser 看看成功否？目前成功。
